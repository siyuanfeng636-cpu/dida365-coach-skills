# 滴答清单 MCP 设置引导

检测到你还没有连接滴答清单，请按下面步骤完成配置。

## 第一步：添加 MCP

```bash
claude mcp add --transport http dida365 https://mcp.dida365.com
```

## 第二步：完成授权

在 Claude Code 或兼容客户端里执行：

```text
/mcp
```

按提示打开授权页面并完成 OAuth。

## 第三步：回到教练流程

授权完成后，告诉我“已连接”或“继续”，我会自动重新检测，并进入以下任一流程：

- 拆长期目标
- 安排时间盒
- 做日复盘
- 做周复盘
- 处理延期或改时间

## 常见问题

Q: 授权页面打不开怎么办？
A: 先检查网络，必要时把链接复制到浏览器手动打开。

Q: 明明配过但还是提示未连接？
A: 再执行一次 `/mcp` 检查状态，通常是授权过期或本地配置文件里没有 `dida365`。
